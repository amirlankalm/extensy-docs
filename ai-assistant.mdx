# Extensy Doc Assistant

Need help with a specific manifest key or a tricky content script? Ask our AI assistant below. It has been trained specifically on this documentation to provide accurate, Extensy-focused guidance.

## How it works

The assistant uses the Google Gemini 1.5 Flash model and a custom system prompt to ensure it only provides information found in these docs.
