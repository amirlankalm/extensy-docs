# The Background Service Worker

The service worker is the extension's main event handler. It remains dormant until an event it's listening for is fired. It is the only place to handle cross-origin API requests and alarm events.

<Tip>
  Use `chrome.runtime.onInstalled` to set up initial state.
</Tip>
