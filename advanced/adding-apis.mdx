# Advanced API Integration

Extensions built with Extensy can interact with almost any public API. 

## Best Practices

1. Use `fetch` in your background service worker for cross-origin requests.
2. Store your API keys securely (e.g., using `chrome.storage.local` or environment variables).

<Note>
  Direct API calls from `content.js` may be blocked by a website's Content Security Policy (CSP).
</Note>
