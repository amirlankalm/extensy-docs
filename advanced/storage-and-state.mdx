# Extension Storage APIs

Use `chrome.storage.local` to store settings and persistent binary data. 

## Syncing Data

Data stored in `local` is not synced across devices. Use `sync` for small configuration data (limit: 100KB).

<Tip>
  You can listen to `chrome.storage.onChanged` to keep your popup and content scripts in sync.
</Tip>
