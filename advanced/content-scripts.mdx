# Injecting Content Scripts

Content scripts are files that run in the context of web pages. They can read details of the web pages the browser visits, and they can make changes to them.

<Note>
  Content scripts cannot use the full power of the `chrome.*` APIs. Use message passing to talk to the background script.
</Note>
